# ContentCreator

![Screenshot of the ContentCreator in use](/assets/ContentCreator.png)

Little helper script to quickly create simple font graphics for things like social media, without interrupting your workflow by having to open up some layout software.

---

Utilizing the [Drawbot module for RoboFont](https://github.com/typemytype/drawBotRoboFontExtension) and [Vanilla](https://github.com/robotools/vanilla/)
